What is GSA?

The GSA File Storage Cloud is the standard technology for file storage across IBM. GSA File Storage Cloud provides an enterprise file system where employees and teams can share data.

With GSA Cloud, end users, projects, and applications can:

  • share files with other users and within teams
  • make backup copies of critical files
  • control access to files using access control lists and groups
  • manage storage with self-service web and command line tools
  • access files through several protocols:
    1. Web - lightweight file access and documentation
    2. Windows Networking - use GSA Cloud like a local hard drive
    3. NFS - for UNIX connectivity to GSA Cloud
    4. SFTP - secure file transfer to and from GSA Cloud

To get started with GSA Cloud, see the New User Guide.
